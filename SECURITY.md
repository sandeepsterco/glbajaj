# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please **do not create a public issue**. Instead, report it privately so we can address it safely.

Please contact us via:

- Email: [**info@stercodigitex.com**](mailto:info@stercodigitex.com)

Include the following information in your report:

- Description of the vulnerability
- Steps to reproduce the issue
- Potential impact
- Suggested mitigation (optional)

We will respond as soon as possible and acknowledge your report.

## Supported Versions

We actively maintain and provide security fixes for the following versions:

| Version | Supported          |
|---------|------------------|
| 1.x     | ✅ Yes            |
| 2.x     | ✅ Yes            |
| <1.0   | ❌ No longer supported |

## Security Updates

Security updates will be communicated via our repository and relevant release notes. Please ensure you stay updated with the latest stable version.

## Responsible Disclosure

We appreciate responsible disclosure and will give credit to researchers who report valid security issues.
